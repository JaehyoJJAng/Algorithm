# 강의 수강용 연습 문제 

<p>
정수를 담은 이차원 리스트, mylist 가 solution 함수의 파라미터로 주어집니다. mylist에 들은 각 원소의 길이를 담은 리스트를 리턴하도록 solution 함수를 작성해주세요.
</p>

```
<제한 사항>
mylist의 길이는 100 이하인 자연수입니다.
mylist 각 원소의 길이는 100 이하인 자연수입니다.

=== 입출력 예 ===
input			output
[[1], [2]]		[1,1]
[[1, 2], [3, 4], [5]]	[2,2,1]
```

<p>입출력 예 설명</p>
